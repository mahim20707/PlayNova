🎮 PlayNova – Smart Offline Learning Game

📌 About the Project

PlayNova is an innovative offline learning game designed for kids (ages 4–10) to help them learn important subjects like Math, English, and Science in a fun and engaging way.

The main goal of this project is to solve real-world problems such as:

- Lack of internet access in rural areas
- Kids getting addicted to non-educational mobile games
- Difficulty in early childhood learning

---

🎯 Features

- 🎮 Game-based learning (Learn while playing)
- 📴 Works without internet (Offline mode)
- 🧠 Smart difficulty system (Easy → Hard)
- 🎁 Reward system (Coins, badges, surprises)
- 🗣️ Voice-based guidance for kids
- 🌈 Kid-friendly UI (Colorful and simple design)
- 📊 Progress tracking system

---

💡 How It Works

1. User starts the game
2. Selects a level (Math / English / Science)
3. Solves questions through fun gameplay
4. Gets rewards for correct answers
5. Unlocks new levels and improves skills

---

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: SQLite
